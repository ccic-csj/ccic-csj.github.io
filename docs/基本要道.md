# 基本要道

## 課程簡介

這福音本是神的大能，要救一切相信的，先是猶太人，后是希腊人。因為神的义正在这福音上顯明出來；這义是本於信，以至於信，如經上所記：「义人必因信得生。」(羅馬書1：16-17)

福音是神的大能。福音不仅仅是个人得救的开始，也是基督徒生活的力量。本學期的基本要道課程以基督徒个人生命為主題，討論因信稱义，个人生命的得救與更新；因信得生，个人生命在家庭、教会，和社会生活中的操練與成長；屬靈操練，學習操練禱告，靈修，團契，敬拜个人和群體的屬靈操練。

基本要道課程系統地講解聖經中最基本的道理，使學習者對真理有整體且清晰的認識。課程以聖經為本，信仰與生活相結合，講課與討論相結合，注重信仰的體驗與操練，適合未信的福音朋友瞭解真理，認識基督，信主的弟兄姐妹鞏固信仰，裝備自己，生命長進。歡迎大家一起參加。
