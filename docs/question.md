# 討論問題

### 討論由組長來組織

=== "週六早餐時間"

    ``` markdown
    1. 請介紹你的名字來由，出生城市，怎麼會來到灣區。對本次退修會的期望
    2. 如果用一個動物或物品來代表自己，你會選擇什麼？ 為什麼？
    3. 請選擇一個隊名，想出一個團隊slogan, 我們是最***的隊 （可以自帶音響效果）
    4. 請想一個能代表你團隊的 dance move
    ```

=== "週六午餐時間"

    ``` markdown
    1. 從剛剛活動的過程當中，請分享你觀察到坐在你左手邊的弟兄或者姐妹的三個優點或特質
    2. 請分享經過這些遊戲，弟兄或者姐妹和你原本印象中不一樣的地方 （任意一個）
    3. 請分享對你生命影響最深的一個人，可以信主前或者信主後
    4. 請分享每一個十年，什麼事情讓你最快樂，什麼事情讓你最焦慮 （十年到八十年）
    5. 如果別人想要鼓勵你，他/她應該說些什麼， 做些什麼？
    ```

=== "週六晚餐時間"

    ``` markdown

    討論如何表演skit

      1. Acts 12, 彼得出监 Peter set free from prison 
      2. Daniel 6, 但以理在狮子坑 Daniel in the pit of lions
      3. Luke 10, 好撒玛利亚人 the good Samaritan
      4. Genesis 24, 以撒娶妻 A wife for Isaac
      5. John 11, 拉撒路复活 Lazarus is brought to life
      6. Esther 4 & 5, 以斯帖皇后救了她的百姓 Queen Esther saved her people
      7. Matthew 14, 22-32, 耶稣水上行走 Jesus walks on the water
      8. Exodus 2, 摩西出生 the birth of Moses 
      9. Matthew 4, 耶稣呼召四渔夫 Jesus calls four fishermen
    ``` 

=== "週日早餐時間"

    ``` markdown
    1. 分享你之前不熟悉，但是通過這次退修會，印象最深的一位弟兄或者姐妹
    2. 分享你在教会（现在或者以前），经历过的最开心的和不开心的事情
    3. 分享你觉得我们在教会里是属于同一个身子吗？为什么？
    4. 作為我們教會，我們如何能做到真正在基督裏的合一？
    5. 在退修會之後，我們能繼續採取什麼行動？
    6. 這次退修會有沒有達到你的期望？你最大的得到是什麼
    7. 討論小組在信息後分享結構，形式，分工等等
    ```
