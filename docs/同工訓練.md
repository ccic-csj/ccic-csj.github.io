# 同工訓練

## 課程簡介

裝備新任的小組長與教會同工，按著神所賜給個人的呼召與恩賜，作忠心的管教。課程將涵蓋聖經對服事的人之教導，認識基督徒會堂與團契，並且訓練同工，在帶領小組查經與關懷肢體上，實際的原則與操練。敬邀約書亞團契的小組長們，與願意參與教會事工的弟兄姐妹們，一同來參加！
